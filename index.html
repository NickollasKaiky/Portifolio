Portfólio Acadêmico

Este site é um portfólio acadêmico desenvolvido para a disciplina de Desenvolvimento Web, ministrada pelo professor Jefferson Chaves.

O objetivo deste portfólio é apresentar os conhecimentos adquiridos em HTML, CSS e JavaScript, além de reunir os projetos desenvolvidos ao longo das aulas.

Sobre a disciplina

  • Fundamentos da arquitetura web
  • Estruturação de páginas com HTML
  • Estilização com CSS
  • Interatividade com JavaScript
  • Publicação de sites
